# Gym Management System Website
This is a simple **Gym Management System website** made using **HTML and CSS**.  
It is created as part of **Lab 5** for the course **Introduction to Computing (SE-105L)**.
This project is for practice and learning basic web development.
## Pages Included
- **Home Page**
- **About Us Page**
- **Membership Page**
Each page is connected using a navigation menu.
## Tools Used
- HTML
- CSS
## Files in Project
home.html
about.html
membership.html
style.css

## How to Open the Website
1. Download or clone the project
2. Open the folder
3. Double click on **home.html**
4. The website will open in your browser
## Purpose of Project
- To learn basic HTML structure
- To use CSS for styling
- To create multiple web pages
- For lab assignment submission
